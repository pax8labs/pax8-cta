# AgentSync Rename — Naming Options for Legal Review

**Date:** March 16, 2026
**Prepared by:** Josh Dulberger
**Context:** Open source release of a CLI tool that deploys Copilot Studio agents across MSP customer tenants. Current working name "AgentSync" conflicts with AgentSync, Inc. (agentsync.io, insurance/HR compliance, $150M+ raised). Need a new name cleared for open source publication under `pax8-oss` GitHub org and `@pax8` npm scope.

---

**Concept:** "Ship" is the universal developer verb for deployment ("ship it"). Combined with "Studio," it directly signals Copilot Studio deployment. Fits the existing shipping metaphors in the codebase. `studioship deploy` reads naturally.

## Summary

| Name              | Risk Level | Memorable | Product Fit                                      | Notes                                                                                  |
| ----------------- | ---------- | --------- | ------------------------------------------------ | -------------------------------------------------------------------------------------- |
| **StudioShip**    | Low        | High      | High — "ship" is the developer verb for deploy   | Zero conflicts found anywhere                                                          |
| **StudioPort**    | Low        | High      | High — plays on import/export CLI commands       | Defunct 1990s audio product, no active claims                                          |
| **ProliferAgent** | Medium     | High      | Medium — new metaphor, distinctive               | Proliferate.com (AI agent platform) is in adjacent space                               |
| **StudioSync**    | Medium     | High      | High — directly references Copilot Studio + sync | WordPress "Studio Sync" feature exists; "Studio" is in Microsoft's trademark portfolio |
| **StudioCast**    | Low-Medium | High      | High — broadcast metaphor                        | Defunct Studiocast entity needs confirmation                                           |
| **StudioRelay**   | Low        | Medium    | High — relay/handoff concept                     | Relay Studio (Danish creative agency) is reversed word order, different industry       |

**Recommendation for legal:** StudioShip, StudioPort, and StudioRelay carry the least risk — no conflicts found in any public registry. ProliferAgent is the most distinctive but needs assessment against Proliferate (active AI agent company). All Studio-prefixed names should be checked against Microsoft's trademark guidelines for "Studio" (Copilot Studio, Visual Studio), though using "Studio" in the name of a tool that deploys _Copilot Studio_ agents may actually be welcomed by Microsoft as ecosystem support. Request legal run a formal trademark clearance search on whichever 2-3 names the team prefers.
